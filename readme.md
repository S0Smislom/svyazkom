# Svyazkom

## Installation and Usage

Create .env and worker/.env files, run the following:

```
    $ touch .env && vim .env
    $ touch worker/.env && vim worker/.env
```



With docker installed, run the following:

```
    $ git clone git@github.com:S0Smislom/svyazkom.git
    $ cd svyazkom
    $ docker compose up
```
